The full Kyaraben app is supported on systems with access to a Linux desktop. For other devices, see [Integrations](#integrations).

## Desktop and Steam Deck

Download the latest AppImage from the [GitHub releases page](https://github.com/fnune/kyaraben/releases) and double-click it to run.

On Steam Deck, switch to Desktop Mode first (STEAM button > Power > Switch to Desktop).

:::note
x86_64 is fully supported. ARM64 is experimental and untested.
[Report issues](https://github.com/fnune/kyaraben/issues) or help test.
:::

## Headless server (CLI only)

For servers without a display, download the CLI binary directly:

```bash
curl -fsSL https://github.com/fnune/kyaraben/releases/latest/download/kyaraben-linux-x86_64 -o kyaraben
chmod +x kyaraben
sudo mv kyaraben /usr/local/bin/
```

See the [headless server guide](/guides/headless-server/) for setup instructions.

## Integrations

Kyaraben syncs with guest devices that run their own emulators. More integrations are planned.

### NextUI handhelds

Download `Kyaraben.pak` from the [GitHub releases page](https://github.com/fnune/kyaraben/releases) and extract it to the `Tools/` directory on your SD card. See [NextUI integration](/nextui/) for supported devices and configuration.
