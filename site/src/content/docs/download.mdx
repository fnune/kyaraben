## Desktop and Steam Deck

Download the app and double-click it to run it:

<LinkCard
  title="Desktop app (x86_64 AppImage)"
  href="https://github.com/fnune/kyaraben/releases/latest/download/Kyaraben-x86_64.AppImage"
/>

On Steam Deck, switch to Desktop Mode first (STEAM button > Power > Switch to Desktop).

## Headless server

Download the CLI binary:

<LinkCard
  title="CLI binary (x86_64)"
  href="https://github.com/fnune/kyaraben/releases/latest/download/Kyaraben-CLI-linux-amd64"
/>

```bash
curl -fsSL https://github.com/fnune/kyaraben/releases/latest/download/Kyaraben-CLI-linux-amd64 -o kyaraben
chmod +x kyaraben
sudo mv kyaraben /usr/local/bin/
```

See the [headless server guide](/guides/headless-server/) for setup instructions.

## Integrations

Kyaraben syncs with guest devices that run their own emulators. More integrations are planned.

### NextUI handhelds

Download, extract, and copy the `Kyaraben.pak` folder to `Tools/<platform>/` on your SD card. See [NextUI integration](/nextui/) for configuration.

<LinkCard
  title="Kyaraben-NextUI.zip"
  href="https://github.com/fnune/kyaraben/releases/latest/download/Kyaraben-NextUI.zip"
/>

Platforms: TrimUI Brick (tg5040), TrimUI Smart Pro (my355), Miyoo Mini (miyoomini), Anbernic RG35XX Plus (rg35xxplus), Miyoo A30 (my282).
