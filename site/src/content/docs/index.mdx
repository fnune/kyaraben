Kyaraben sets up emulators on Linux so you can play retro games. It works on
any Linux distribution, including the Steam Deck.

## What it does

<CardGrid>
  <Card title="Select and apply">
    Choose which gaming systems you want to emulate. Kyaraben installs the emulators and generates
    their configuration. One click.
  </Card>
  <Card title="Unified folder structure">
    ROMs, BIOS files, saves, and screenshots organized in a single `~/Emulation` directory. Every
    emulator points to the same place.
  </Card>
  <Card title="Observability">
    Kyaraben tracks what it installs, shows diffs before changing config files, and avoids littering
    your system. You always know what changed and why.
  </Card>
  <Card title="Clean by design">
    No system packages modified, no root access needed. Everything lives in user directories.
    Uninstall removes only what kyaraben put there.
  </Card>
</CardGrid>

## How it works

1. Download the AppImage and run it
2. Select the systems you want to emulate
3. Click apply
4. The UI shows which BIOS or firmware files are needed
5. Drop your ROMs into the created folders and play

## What kyaraben promises

- Installations are reproducible: same config, same result, every time
- Your system is never polluted: no root access, no system packages modified
- Kyaraben installs itself, its emulators, and frontends to `~/.local/bin/`
  and `~/.local/share/kyaraben/`, with `.desktop` entries for everything
- Uninstall is clean: your ROMs, saves, and BIOS files are never touched
- Your emulator settings are preserved: kyaraben manages specific keys and
  leaves the rest alone
- You see what changes before it happens: config diffs are shown before apply

See [app reference](/using-the-app/) for the full list of guarantees, non-goals,
and future plans.
