Kyaraben sets up emulators on Linux so you can play retro games. It works on
any Linux distribution, including the Steam Deck.

## How it works

1. Download the AppImage and run it
2. Select the systems you want to emulate
3. Click apply
4. Drop your ROMs into the created folders and play
