## How it works

Kyaraben uses Nix to install emulators reproducibly and in isolation. You never
see or interact with Nix directly. It is an implementation detail.

1. Download the AppImage and run it
2. Select the systems you want to emulate
3. The UI tells you which BIOS or firmware files are needed
4. Click apply
5. Drop your ROMs into the created folders and play

## Guarantees

Kyaraben makes specific promises about how it behaves on your system. See
[how it works](/how-it-works/) for the full list.

- Installations are reproducible: same config, same result, every time
- Your system is never polluted: no root access, no system packages modified
- Uninstall is clean: your ROMs, saves, and BIOS files are never touched
- Your emulator settings are preserved: kyaraben manages specific keys and
  leaves the rest alone
- You see what changes before it happens: config diffs are shown before apply
