Kyaraben is free software built on other free software. If you find it useful, consider supporting the projects that make it possible.

## Supporting Kyaraben

- [Sponsor on GitHub](https://github.com/sponsors/fnune)

You can also help by reporting bugs, contributing code or documentation, and sharing Kyaraben with others.

## Upstream projects

These are the projects Kyaraben depends on. Supporting them helps the entire emulation ecosystem. Projects are grouped by category.
