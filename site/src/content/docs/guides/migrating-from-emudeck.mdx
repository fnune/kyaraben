---
title: Migrating from EmuDeck
description: Import your existing EmuDeck collection into Kyaraben
---

:::caution[Work in progress]
The import tool is under active development. Back up your collection before migrating.
:::

Kyaraben can scan your existing EmuDeck directories and show you what you need to copy over: ROMs, BIOS files, saves, and EmulationStation DE metadata. Files are not moved automatically; the tool shows you what exists and where Kyaraben expects it.

In the app, go to Settings > Import. From the CLI, run `kyaraben import scan ~/Emulation`. See [CLI reference](/using-the-cli/#kyaraben-import-scan) for output options.
